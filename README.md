# Joe Severino

Cybersecurity graduate student at Georgia Tech focused on network security, infrastructure, and practical security tooling.

I use this GitHub to document and publish projects related to security engineering, networking, automation, and homelab infrastructure. Most of my work is built around real systems I run myself, including WordPress hardening, TLS automation, private PKI, DNS filtering, reverse proxy management, and local infrastructure.

## Featured Projects

- **Severino Labs Security Layer** — Custom WordPress security plugin for application hardening, file integrity monitoring, security event logging, browser security headers, and passkey-first login customization.

- **TLS Cert Renewal Automation** — Automates Let’s Encrypt wildcard certificate renewal and cPanel deployment using Certbot, Cloudflare DNS validation, and cPanel UAPI.

- **Zsh Git Deploy Workflow** — Reusable deployment workflow with GitHub integration, SSH deploy keys, signed local commits, and fast-forward-only production pulls.

- **Caddy Manager** — CLI tool for managing Caddy reverse proxy sites, automatic TLS snippets, and config backups.

- **Homelab Certificate Generator** — Issues TLS certificates from a private root CA hosted on an offline VM.

- **Study Quiz** — Client-side quiz app that imports a local JSON question bank, tracks progress, and avoids storing course content on a server.
